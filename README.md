# League Dashboard

Обновлённый дашборд рейтинга для GitHub Pages.

## Что улучшено

- Пересобран публичный экран рейтинга с упором на визуал:
  - новый hero-блок;
  - акцентный блок лидеров;
  - более читаемая лента участников;
  - улучшенная модалка с правилами начисления баллов.
- Сохранена текущая логика загрузки, синхронизации и админ-панели.
- Убрана сборка в single-file режим, из-за чего сайт стал заметно легче для GitHub Pages.
- Исправлены TypeScript-проблемы:
  - добавлен `shouldSum` в тип параметра;
  - добавлен `vite-env.d.ts` для импортов изображений;
  - унифицированы константы из `src/config.ts`.

## Команды

```bash
npm ci
npm run dev
npm run build
```

## Деплой

GitHub Actions уже настроен через `.github/workflows/deploy.yml`.
После пуша в `main` проект собирается и публикуется на GitHub Pages.

## Важное замечание

Пароль админки сейчас хранится во фронтенде (`src/config.ts`).
Это удобно для текущей схемы, но небезопасно. Если нужен реальный закрытый доступ, лучше вынести авторизацию на бэкенд / Apps Script.
